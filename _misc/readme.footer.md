## Tips

### Off gopwt.Empoer()

If you don't need gopwt or need to stop gopwt, you can!

```
GOPWT_OFF=1 go test -covermode count
PASS
coverage: 60.8% of statements
ok      github.com/ToQoz/gopwt  1.008s
```
